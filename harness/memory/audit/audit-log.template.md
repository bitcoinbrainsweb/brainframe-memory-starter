# Memory Audit Log - <ISO-date>

Append-only. One line per applied verdict. This is a template: real audit logs are
generated by running the audit routine against your own memory. Do not commit real
memory snapshots into a public repo.

Format:

```
<ISO-time> run-id=<run-id> hash=<content-hash> verdict=<verdict> action=<action> status=<success|failed|stale>
```

Example rows (illustrative, not real data):

```
2026-01-01T09:00:00Z run-id=2026-01-01-a1b2c3 hash=a1b2c3d4 verdict=KEEP action=none status=success
2026-01-01T09:00:01Z run-id=2026-01-01-a1b2c3 hash=e5f6g7h8 verdict=REMOVE action=remove status=success
2026-01-01T09:00:02Z run-id=2026-01-01-a1b2c3 hash=i9j0k1l2 verdict=MOVE_TO_FILE action=append+remove status=success
```
